# Mac Setup

Fork the repo and exit the variables in setup.sh
```gpg_key=""
git_dir="${HOME}/git"
git_config_name=""
git_email=""
github_user=""
github_org=    # Leave Empty to Skip.
```


Lookup your GPG key: hkp://pgp.mit.edu

### Notes
- Brew Bundle will skip Applications already installed
